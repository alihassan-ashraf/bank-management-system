# 🏦 Python Bank Management System

A Python-based interactive web application developed using **Streamlit** to manage customer bank accounts. The application allows users to create accounts, deposit money, withdraw money, check balances, search accounts, delete accounts, and view bank summary through a simple web interface.

## 🚀 Live Demo

Try the application here:  
https://bank-management-system-53tfqgsurk6rmr8xuxcidq.streamlit.app/

---

## 📌 Project Overview

The Python Bank Management System simplifies basic banking operations. Users can create customer accounts, manage deposits and withdrawals, check account balances, search account records, and view overall bank statistics in one place.

This project was developed to strengthen my Python concepts by applying them in a practical portfolio project.

---

## ✨ Features

- Create new bank accounts
- Display all accounts
- Search accounts by Account Number
- Deposit money
- Withdraw money
- Check account balance
- Delete accounts
- View total accounts and total bank balance
- Interactive web interface using Streamlit

```bash

cd python-bank-management-system
pip install -r requirements.txt
streamlit run app.py
```
